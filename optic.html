Оптика - Занимательная физика

1. Закон отражения

Закон отражения описывает поведение света при его отражении от поверхности.

Формула закона отражения: θотраж = θпад
Где:
  • θотраж — угол отражения (градусы)
  • θпад — угол падения (градусы)
Пример: Если угол падения света составляет 30°, то угол отражения также будет равен 30°.

2. Закон преломления (Закон Снеллиуса)

Закон преломления описывает, как свет изменяет направление при переходе из одной среды в другую с разными показателями преломления.

Формула закона Снеллиуса: n1 * sin(θ1) = n2 * sin(θ2)
Где:
  • n1, n2 — показатели преломления сред
  • θ1, θ2 — углы падения и преломления (градусы)
Пример: Если свет проходит из воздуха в воду, где показатели преломления n1 = 1 (для воздуха) и n2 = 1.33 (для воды), и угол падения составляет 30°, то угол преломления можно найти по формуле Снеллиуса:

1 * sin(30°) = 1.33 * sin(θ2)
sin(θ2) = sin(30°) / 1.33 ≈ 0.375
θ2 ≈ 22.09°

3. Линза и её фокусное расстояние

Линзы используются для изменения направления световых лучей, а фокусное расстояние определяет, насколько сильно линза изменяет световой поток.

Формула для линзы: 1/f = 1/do + 1/di
Где:
  • f — фокусное расстояние линзы (метры, м)
  • do — расстояние до объекта (метры, м)
  • di — расстояние до изображения (метры, м)
Пример: Если объект находится на расстоянии 50 см от линзы, а изображение получается на расстоянии 100 см, то фокусное расстояние линзы можно вычислить:

1/f = 1/0.5 + 1/1 = 2 + 1 = 3
f = 1/3 ≈ 0.33 м

4. Дисперсия света

Дисперсия — это явление, при котором свет разных длин волн (цветов) преломляется по-разному при прохождении через материал.

Формула для дисперсии: v = c / n
Где:
  • v — скорость света в материале (м/с)
  • c — скорость света в вакууме (приблизительно 3 * 10^8 м/с)
  • n — показатель преломления материала (без единиц)
Пример: Если показатель преломления материала n = 1.5, то скорость света в этом материале будет:

v = 3 * 10^8 м/с / 1.5 = 2 * 10^8 м/с
